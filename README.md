# Система мониторига самолетов
Программа для получения и анализа данных о самолетах, находящихся в воздушном пространстве различных стран, с использованием открытых API.

## Описание проекта
Проект представляет собой консольное приложение для мониторинга воздушного пространства. Программа получает данные о самолетах через открытые API:

- **Nominatim API** (OpenStreetMap) — для получения географических координат стран
- **OpenSky Network API** — для получения данных о самолетах в реальном времени

Полученные данные сохраняются в JSON-файл, что позволяет анализировать их позже без повторных запросов к API.

## Функциональность

- ✅ Получение данных о самолетах по названию страны
- ✅ Отображение всех сохраненных самолетов
- ✅ Вывод топ-N самолетов по высоте полета
- ✅ Фильтрация самолетов по стране регистрации
- ✅ Сохранение данных в JSON-файл
- ✅ Очистка всех сохраненных данных
- ✅ Обработка ошибок и валидация данных


## Инструкции по установке и запуску

Для запуска проекта необходимо клонировать репозиторий, установить зависимости с помощью poetry и запустить скрипт main.py.
